# 기능 명세서

| 서비스 | 우선순위 | 세부항목 | 주요기능 | 요구사항 (설명, 필요 정보) |
|--------|-----------|-----------|----------|-----------------------------|
| 사용자 | P1 | 회원가입 | - 유저 회원가입 버튼 or 링크 제공<br>- 강사, 학생 여부 확인<br>- 공통 form : 이름, 닉네임, 이메일, 비밀번호, 비밀번호 확인, 생년월일<br>- 강사 form : 학사 정보(강사의 졸업, 재학여부 확인(석사, 대학, 고등), 상담의 경우 심리학 석박사 확인), 소속, 강의 전공<br>- 학생 form : 취약계층 확인코드<br>- 가입 버튼 제공 | - 강사, 학생 여부에 따라 다른 입력 양식 제공<br>- 필수 정보: 이름, 닉네임, 이메일, 비밀번호, 생년월일 등<br>- 가입 시 유효성 검사 수행 |
| 사용자 | P1 | 회원 정보 조회 | - 유저 정보 제공<br>- 기본적인 회원 정보 출력<br>- 회원 정보 수정 버튼 제공<br>- 회원 탈퇴 버튼 제공<br>- 강사: 현재까지 진행한 봉사시간 출력 | - 사용자 프로필 화면에서 조회 가능<br>- 필수 정보: 사용자 ID, 회원 가입 정보 등<br>- 강사의 경우 봉사 시간 데이터 포함 |
| 사용자 | P1 | 회원탈퇴 | - 유저 회원 탈퇴 재확인<br>- User Profile 페이지에서 별도 링크 제공.<br>- Modal를 사용하여 “정말 회원을 탈퇴하시겠습니까?” alert 출력.<br>- 예를 누를 시 Usermodel에서 삭제 | - 회원 탈퇴 시 계정과 관련된 모든 데이터 삭제<br>- 필수 정보: 사용자 ID<br>- 회원 탈퇴 요청 확인을 위한 추가 보안 절차 필요 |
| 사용자 | P1 | 로그인 | - 유저 로그인 버튼 or 링크 제공<br>- form : 이름, 아이디<br>- 회원가입 링크, 비밀번호 찾기 링크 삽입. | - 로그인 시 아이디 및 비밀번호 검증<br>- 필수 정보: 사용자 아이디, 비밀번호<br>- 비밀번호 암호화 및 안전한 인증 절차 필요 |
| 사용자, 인증 | P1 | 취약 계층 학생 인증 | - 취약 계층 인증 시스템<br>- 교육청이나 학교에서 취약 계층 학생들에게만 제공되는 인증 코드를 발급해 학생들은 이 코드를 사이트에 입력하여 가입 (사전에 DB에 코드가 등록되어 있어야 함). | - 사전에 발급된 인증 코드와의 매칭 필요<br>- 필수 정보: 인증 코드, 학생 정보<br>- 인증 코드 검증 후 추가적인 회원 가입 절차 진행 |
| 화상교실 | P1 | 내 화면 공유 | - 화면 켜기 끄기 | - 사용자의 화면을 다른 사용자와 공유 가능<br>- 필수 정보: 사용자 ID, 화상 교실 ID<br>- 공유 상태에서의 화면 전환 기능 제공 |
| 화상교실 | P1 | 내 카메라 켜기 | - 내 카메라 켜기 끄기 | - 사용자의 웹캠을 켜고 끌 수 있는 기능<br>- 필수 정보: 사용자 ID, 화상 교실 ID<br>- 카메라 상태 변경에 따른 UI 업데이트 필요 |
| 화상교실 | P1 | 교실 삭제 | - 선생님의 강제 종료 ⇒ 교실<br>- DB에 강의 히스토리 테이블이 있고, 1건의 강의에 대해 1개의 요약 등이 있고, 그 히스토리id를 선생님도 학생도 가지고 있게 | - 강의 종료 시 모든 참여자의 세션 종료<br>- 필수 정보: 화상 교실 ID, 강의 ID<br>- 강의 기록 및 요약 정보 저장 후 교실 삭제 |
| 화상교실 | P1 | 교실 입장 | - 공유링크 or 입장코드를 통해 입장 가능<br>- 사전에 신청한 강의의 교실은 ‘내 강의실’ 에서 입장 가능<br>- 닉네임 등록<br>- 프라이빗하게만 입장가능<br>1. 공유 링크를 통해 입장<br>2. 입장코드를 통해 입장 (퍼블릭은 고려하지 않음) | - 입장 시 닉네임 확인 및 등록 필요<br>- 필수 정보: 교실 입장 코드, 사용자 ID<br>- 입장 시 보안 및 인증 절차 필요 |
| 화상교실 | P1 | 채팅 | - 화상수업 중 채팅 가능 | - 실시간 채팅 기능 제공<br>- 필수 정보: 사용자 ID, 화상 교실 ID<br>- 채팅 내용 저장 및 기록 기능 필요 |
| 화상교실 | P1 | 교실 생성 | - 교실 생성<br>- 공유 가능한 링크or입장코드 생성<br>- 비고: 교실 생성 기능은 이미 지자체/봉사동아리 등에 의한 사전 조율이 된 상황에서 사용 | - 교실 생성 시 고유 링크와 입장 코드 생성<br>- 필수 정보: 강사 ID, 강의 정보<br>- 교실 생성 후 학생에게 링크 공유 가능 |
| 강의 | P1 | 강의 등록하기 | - 선생님 회원이 강의 등록<br>- 강의 계획서 (날짜, 시간 등 타임테이블)<br>- 강사가 강의를 등록할 수 있는 기능을 제공 | - 강의 정보 입력 후 등록<br>- 필수 정보: 강의 제목, 날짜, 시간 등<br>- 강의 등록 후 학생이 신청할 수 있도록 게시 |
| 강의 | P1 | 강의 신청하기 | - 학생 회원이 강의 신청<br>- 강의 상세정보에서 강의 신청하기 버튼을 통해 강의를 신청할 수 있는 기능 제공 | - 학생이 원하는 강의를 선택하여 신청<br>- 필수 정보: 학생 ID, 강의 ID<br>- 신청 후 강의실 입장 링크 제공 |
| 강의 | P1 | 강의 상세정보 조회 | - (비교적) 상세한 강의 커리큘럼 확인 가능<br>- 강의 데이터: 강의 시간, 학생 수, 강의 분야 | - 강의 내용과 커리큘럼 확인 가능<br>- 필수 정보: 강의 ID, 강사 정보<br>- 강의에 대한 추가 정보 제공 |
| 강의 | P1 | 내 강의실 | - 수강 혹은 진행한 강의 정보 제공<br>- 강사: 진행 중인 강의 정보 제공, 진행 중인 강의에 대해 퀴즈 등록 기능 제공<br>- 학생: 수강한 강의 정보 제공, 수강했던 강의 정보를 토대로 진로 추천을 받을 수 있는 버튼 제공<br>- 학생: 수강한 내역에 대해 증명서 발급을 받을 수 있는 버튼 제공 | - 현재 진행 중인 강의와 완료된 강의 정보 제공<br>- 필수 정보: 강의 ID, 학생/강사 ID<br>- 진로 추천 및 증명서 발급 기능 제공 |
| 사용자, 생성형 AI | P2 | 진로 추천 | - 수강한 과목을 토대로 진로 추천<br>- 대분류를 토대로 본인이 관심있던 과목에 대한 레이더 차트 제공 (스탯 측정)<br>- 수강했던 과목들을 토대로 chatGPT가 학생이 진학하면 좋을 진로를 추천 | - 과거 수강 기록을 기반으로 진로 추천<br>- 필수 정보: 학생의 수강 기록, 과목 정보<br>- AI 분석을 통한 맞춤형 진로 제안 |
| AI, 화상교실 | P2 | 강의 참여도 측정 | - 기본 점수에서 경고점수 감점, 퀴즈 점수 가점<br>- 기본점수(70), 졸음/자리비움의 경고점수마다 5점 감점.<br>- 지루함 측정 (지루함+참여도 = 100) | - 강의 참여도에 따른 점수 관리<br>- 필수 정보: 학생 ID, 강의 ID<br>- 졸음 감지 및 참여도에 따른 실시간 점수 조정 |
| 화상교실 | P2 | 집중도 향상 기능 | - 강사가 퀴즈 풀기 기능을 통해 퀴즈 출력<br>- 졸음 혹은 자리비움이 감지되었을 때 각 상황에 맞는 확인창이 출력.<br>- 제한 시간 내에 확인을 누르지 않을 시 경고음과 함께 감점 진행.<br>- 강사가 퀴즈풀기 버튼을 누를 때, 사전에 준비된 퀴즈가 한 문제 출력.<br>- 퀴즈는 정해진 제한 시간 안에 풀어야 함.<br>- 퀴즈를 맞췄을 시, 기본점수에서 (30/문제 개수)만큼 점수 증가. | - 강의 중 집중도를 높이기 위한 퀴즈 기능<br>- 필수 정보: 강의 ID, 학생 ID<br>- 퀴즈 결과에 따른 즉각적인 피드백 제공 |
| 강의 | P2 | 퀴즈 등록 | - 강의 시작 전 사전에 강사가 퀴즈를 등록.<br>- 등록된 퀴즈를 기반으로 추후 퀴즈풀기 기능 수행 시 퀴즈가 출력될 수 있게 준비. | - 강사가 미리 퀴즈를 등록<br>- 필수 정보: 강의 ID, 퀴즈 내용<br>- 퀴즈는 강의 중에 사용할 수 있도록 준비 |
| 강의 | P2 | 수강내역 저장 | - 추후 chatGPT 진로 추천을 위하여 학생이 수강한 강의 제목과 대분류를 저장 | - 수강 내역을 데이터베이스에 저장<br>- 필수 정보: 학생 ID, 강의 ID, 수강 날짜<br>- 진로 추천을 위한 데이터로 활용 |
| 강의 | P2 | 인기강의 조회 | - 인기 강의를 조회 | - 인기 있는 강의 리스트 제공<br>- 필수 정보: 강의 ID, 수강 학생 수<br>- 조회 수 및 신청자 수에 기반한 인기 강의 랭킹 제공 |
| 강의 | P2 | 강의 참여도 저장 | - 매 수업 종료되었을 때, 학생이 참여한 적극성을 정량적 데이터로 변경하여 저장 | - 학생의 강의 참여도 데이터를 저장<br>- 필수 정보: 학생 ID, 강의 ID, 참여도 점수<br>- 저장된 데이터를 기반으로 성취도 평가 |
| 퀴즈 | P3 | 퀴즈 점수 매기기 | - 퀴즈 점수 매기기<br>- 미리 등록한 답변을 통해 퀴즈를 자동으로 채점 | - 퀴즈 완료 후 자동 채점 기능 제공<br>- 필수 정보: 퀴즈 ID, 정답 데이터<br>- 점수는 시스템에 자동으로 저장 |
| 강의, 생성형 AI | P3 | 질의응답 기능 | - 학생들이 수업 중 모르는 내용을 물어볼 수 있는 간단한 생성형 AI 기능 제공 | - AI 기반의 실시간 질의응답 기능<br>- 필수 정보: 질문 내용, 강의 ID<br>- 학생의 질문에 대한 즉각적인 답변 제공 |
| 강의 인증 | P5 | 학생 강의 수료 인증 | - 내 강의실 또는 인증 페이지에서 한 커리큘럼에 대해 SSAFY처럼 학습시간을 표로 뽑아서 인증서 생성 | - 강의 완료 후 수료 인증서 발급<br>- 필수 정보: 학생 ID, 수료 강의 ID<br>- 인증서는 PDF 형식으로 제공 |
| 강의 인증 | P5 | 강사 강의 교습 인증 | - 상동이나 교육을 제공한 것으로 인증서 생성 | - 강사가 제공한 강의에 대한 인증서 발급<br>- 필수 정보: 강사 ID, 강의 ID<br>- 인증서는 강사 프로필에 저장 |
